# 🥷🆚👮‍♂️ 도둑과 경찰: 비대칭 멀티플레이 액션 게임

> 3인칭 시점에서 펼쳐지는 **비대칭 PvP 멀티플레이 게임**  
> **도둑**은 금괴를 모아 탈출을, **경찰**은 도둑을 모두 체포하는 것이 목표입니다.

---

## 🎮 1. 게임 장르

**3인칭 멀티플레이 비대칭 PvP 액션 게임**

- Third-Person View 기반
- 실시간 멀티플레이
- 비대칭 역할 구조 (경찰 vs 도둑)
- 아이템 기반 스킬 & 전략적 협력 요소 포함

---

## 🧭 2. 게임 개요

### 경찰과 도둑, 상반된 목표와 전략  
각기 다른 승리 조건을 가진 두 진영 간의 경쟁이 벌어집니다.

#### 🔷 경찰 (Cops)
- **목표:** 모든 도둑을 체포하여 감옥에 가두면 승리
- **플레이 방식:**
  - 도둑 추적 및 체포
  - 감시, 협공, 경로 차단 등 전략적 맵 활용
  - 속도 저하 투사체 스킬로 도둑 견제

#### 🔶 도둑 (Thieves)
- **목표:** 금괴 아이템을 수집하여 목표 점수 도달 후 탈출하면 승리
- **플레이 방식:**
  - 은신, 텔레포트 등 전략적 스킬 사용
  - 체포된 팀원은 감옥 앞에서 **3초간 F키 입력으로 구출 가능**
  - 팀워크 기반의 회피 및 생존 중심 플레이

## 🧩 3. 주요 시스템 요약

| 시스템 | 설명 |
|--------|------|
| **역할** | 비대칭 역할 (경찰 vs 도둑) |
| **시점** | 3인칭 시점 (TPV) |
| **스킬** | 아이템 기반 능력 (은신, 텔레포트, 속도 증가/감소 등) |
| **체포 & 구출** | 경찰에게 잡히면 감옥에 감금 → 팀원이 구출 가능 |
| **투사체** | 조준 후 발사 → 명중 시 대상 이동 속도 감소 |
| **속도 스킬** | 사용 시 일시적으로 이동 속도 증가 |
