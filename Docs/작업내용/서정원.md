# 멤버 서정원 작업 내용 상세 보고

## 주요 기여 내용 요약
- **GAS 확장:** 도둑/경찰의 주요 행동(잡기, 체포, 은신, 이동 등)을 Gameplay Ability System 기반으로 설계 및 구현  
- **감옥 시스템:** 감옥 내 상태 관리, 충돌 및 벽 제어 로직, 해방 이펙트 구현  
- **UI 시스템:** HUD, 미니맵, 결과창, 팀 정보 등 주요 UI를 GameState 및 ASC 이벤트와 연동  
- **AI 제어:** AI 무작위 이동 및 정지 어빌리티를 GAS 구조에 통합  
- **디메리트 시스템:** AI 체포 시 패널티 적용 로직으로 핵심 밸런스 구현  

---

## 1. 게임플레이 어빌리티 시스템 (GAS)

### 1.1 ReleaseAbility (풀어주기 어빌리티)
- **기능:** 감옥에 잡힌 도둑을 3초간 키 입력으로 해방시키는 기능 구현, UI 프로그레스 바와 실시간 연동  
- **주요 구현 사항:**  
  - 감옥 근처 접근 시 상태 태그 부여  
  - F키 입력 시 서버 권한으로 어빌리티 실행  
  - 3초 대기 후 성공 시 감옥 벽 비활성화 및 도둑 해방  
  - 조기 입력 해제 시 즉시 중단  
  - UI 프로그레스 바와 GameplayEvent 연동  

### 1.2 RandomMoveAbility (AI 무작위 이동)
- **기능:** AI가 일정 반경 내에서 무작위 이동 후 대기 시간을 두고 반복 이동  
- **특징:**  
  - AI 전용 이동 루프 구조  
  - 이동 종료 후 랜덤 Delay  
  - 어빌리티 종료 시 즉시 정지  

### 1.3 ArrestedAbility / StopMovementAbility (체포 관련)
- **ArrestedAbility:** 경찰이 도둑을 감옥 위치로 이동시키는 기능  
- **StopMovementAbility:** 입력 및 화면 회전 차단, AI는 RandomMoveAbility 중단  

### 1.4 InvisibilityAbility (은신 어빌리티)
- **기능:** 도둑이 3초간 은신  
- **구성:**  
  - HideEffect: 3초 지속  
  - 태그: 은신 상태  
  - GameplayCue: 투명도 제어  
- **효과:**  
  - 시작 시 투명도 감소  
  - 종료 시 투명도 복원  
- **쿨타임:** 은신 쿨타임 태그 적용  

### 1.5 TeleportAbility (텔레포트 어빌리티)
- **기능:** 도둑이 전방 일정 거리로 순간이동  
- **주요 로직:**  
  - 전방 라인트레이스를 통해 충돌 지점 계산  
  - 충돌 시 약간 뒤로 위치 보정  
  - 땅에 끼이지 않도록 2차 Ground Trace 적용  
  - 이동 후 쿨타임 태그 적용

## 2. 감옥 시스템

### 2.1 구조 및 충돌
- Box Collision 2개: 잡힘 유지용 / 구출 시도용  
- 오버랩 이벤트를 통해 태그 및 이펙트 제어  
- 감옥 벽 3초간 일시 비활성화 기능  

### 2.2 주요 로직
- 감옥 벽 숨기기 → Mesh 비활성화, Collision 제거, 일정 시간 후 복구  
- 풀기 가능 상태 처리 → ReleaseAbility와 연동  
- 서버에서 감옥을 GameState에 등록

## 3. UI 시스템

### 3.1 ToolTipWidget (팀 정보)
- 팀 아이콘 및 설명 표시  
- 비동기 리소스 로딩 후 아이콘 적용  
- Police / Thief 팀별 정보 표시  

### 3.2 ResultWidget (게임 결과)
- 승패 여부 및 조건 표시  
- PlayerState의 팀 태그 기반으로 결과 판정  
- 예시

| 조건               | 문구                       |
|-------------------|---------------------------|
| CatchAllThief      | 모든 도둑이 잡혔습니다.    |
| ThiefWinByScore    | 도둑이 모든 포인트를 모았습니다. |

### 3.3 PlayingWidget (HUD)
- 실시간 점수, 도둑 수, 남은 시간 표시  
- GameState 델리게이트 기반 자동 갱신  
- 바인딩 함수: 점수, 도둑 수, 게임 시간 변경 시 UI 업데이트  

### 3.4 MiniMapWidget (미니맵)
- 플레이어, 팀원, 아이템, 감옥 위치 실시간 표시  
- 월드 좌표 → 미니맵 좌표 변환  
- 델리게이트 기반 자동 갱신  

---

## 4. 디메리트 시스템 (패널티 로직)
- **기능:** AI 체포 시 플레이어에게 일시적 행동 불가 상태 부여  
- **세부 구현:**  
  - GameMode의 체포 함수 반환값으로 AI/플레이어 구분  
  - AI 체포 시 패널티 적용  
  - 이동, 스킬 사용, 시점 제어 불가 상태 적용  
  - GameplayCue를 통한 시각 연출
